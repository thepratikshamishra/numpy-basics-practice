# 🔢 NumPy Basics Practice

A beginner-friendly collection of Jupyter notebooks to learn and practice **NumPy** — the foundational library for numerical computing in Python.

---

## 📓 Notebook

| File | Description |
|------|-------------|
| `numpy_fundamentals.ipynb` | Core NumPy concepts with hands-on examples and exercises |

---

## 📌 Topics Covered

- Creating NumPy arrays (1D, 2D, 3D)
- Array properties — `shape`, `ndim`, `dtype`, `size`
- Array indexing and slicing
- Array reshaping and transposing
- Mathematical and statistical operations
- Broadcasting
- Universal functions (ufuncs)
- Random number generation with `np.random`

---

## 🚀 Getting Started

### Prerequisites

Make sure you have Python and the following packages installed:

```bash
pip install numpy jupyter
```

### Run the Notebook

```bash
git clone https://github.com/thepratikshamishra/numpy-basics-practice.git
cd numpy-basics-practice
jupyter notebook numpy_fundamentals.ipynb
```
